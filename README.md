# dotfiles

Repo for versioning and setting up dotfiles.


# Usage

```sh
git clone https://github.com/hgrif/dotfiles.git
cd dotfiles
./makesymlinks.sh
```

All files/folders in folder `files/` are symbolic linked to their `.`-equivalents in `~`. Existing files are backed up.
